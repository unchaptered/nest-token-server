## Stack

본 프로젝트는 다음과 같은 기술 스텍을 포함하고 있습니다.

1. Nest.JS
2. PostgreSQL, pgAdmin, TypeORM
3. dotenv, cross-env, @nest/dotenv
4. passport-jwt, @nest/passport-jwt, @nest/jwt
5. bcrypt
6. joi, class-validator, class-transformer
7. uuid
